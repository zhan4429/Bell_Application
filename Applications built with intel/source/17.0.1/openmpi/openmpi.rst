Openmpi
==============================

Description
~~~~~~~~
An open source Message Passing Interface implementation.

Versions and Dependencies
~~~~~~~~
- 2.1.6
- 3.1.4
   #. zlib/1.2.11

- 4.0.5
   #. zlib/1.2.11

Module
~~~~~~~~
You can load the modules by::

    module load openmpi
